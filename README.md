This portfolio is designed to:
✅ **Showcase my professional profile** with a clean, modern design

✅ **Highlight my skills** in programming languages, frameworks, and tools

✅ **Display my academic and professional projects** with detailed descriptions

✅ **Provide easy contact options** for potential employers and collaborators

Perfect for **software engineers, students, and professionals** looking to present their work in a polished, interactive format.

## 📦 Installation

### **Prerequisites**
Ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v18 or higher)
- [Angular CLI](https://angular.io/cli) (v17 or higher)

### **Quick Start**

1. **Clone the repository**
   ```bash
   git clone https://github.com/kerfaiyass54/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   ng serve
   ```
   Open [http://localhost:4200](http://localhost:4200) in your browser.

4. **Build for production**
   ```bash
   ng build --configuration production
   ```

### **Alternative Installation Methods**

#### **Using Docker**
If you prefer Docker, follow these steps:
1. Ensure Docker is installed on your system.
2. Build and run the Docker container:
   ```bash
   docker build -t portfolio .
   docker run -p 4200:80 portfolio
   ```

#### **Development Setup**
For a full development environment:
1. Install Angular CLI globally:
   ```bash
   npm install -g @angular/cli@latest
   ```
2. Follow the standard installation steps above.


## 🔧 Configuration

### **Environment Variables**
- Configure environment-specific settings in `src/environments/environment.ts` and `environment.prod.ts`.

### **Firebase Hosting**
- Deploy to Firebase using the `firebase.json` configuration:
  ```bash
  npm install -g firebase-tools
  firebase login
  firebase init hosting
  firebase deploy
  ```

### **Customization Options**
- **Change colors**: Modify `src/styles.css`
- **Update animations**: Edit CSS keyframes in respective component files
- **Add new sections**: Create new components and update `app.component.html`
